Engineering Architecture

System Blueprint

  • React Client: Handles UI states, WebSocket streams, and active Monaco code editor.
  • Node.js Server: Manages room Auth, validations, and proxies RCE (Remote Code Execution) tasks.
  • MongoDB Cluster: Stores secure user profiles, algorithm problems, and session histories flexibly.
  • Inngest Workers: Asynchronous task polling for heavy execution queues avoiding main thread locks.

Computer Science Theory

The Security Sandbox

Concept: Remote Code Execution (RCE) is a major server vulnerability if users can run malicious system scripts.

  • Isolation Layer: The backend pipes the candidate's code into a strict Docker Container / Sandbox limits execution to 2 seconds and 50MB RAM.
  • Privilege Drop: The runtime operates without disk access or network access outward.
  • Results Parsing: Only raw stdout/stderr streams are extracted and returned sequentially over the REST API connection.
Networking

WebRTC: Real-Time Protocol

  • Concept: Traditional HTTP 1.1 polling is vastly too slow for live video pipelines.
  • Implementation: Uses UDP WebRTC data channels for direct peer-to-peer data flow avoiding heavy server routing.
  • Synchronization: Ensures video framing updates, chat events, and editor cursors sync in under 50ms globally.
